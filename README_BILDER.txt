README_BILDER.txt

In diesem Projekt werden keine lokal gespeicherten Porsche-Fotos verwendet.
Grund:
Die vorgegebene Projektstruktur enthält im img-Ordner nur diese README-Datei.

Die echten Fahrzeugfotos werden in den HTML-Dateien extern über Wikimedia Commons eingebunden.
Das ist besser nachvollziehbar als Bilder aus Google oder Instagram zu kopieren.

Wichtig:
- Nicht einfach Bilder von Google speichern.
- Keine Instagram-Bilder verwenden.
- Keine offiziellen Porsche-Pressebilder ohne Erlaubnis verwenden.
- Jede Wikimedia-Commons-Datei hat eine eigene Lizenz.
- Vor einer echten Veröffentlichung müssen Autor, Lizenz und Quelle korrekt angegeben werden.
- Die genaue Bildquellenliste steht auf quellen.html.

Logo-Hinweis:
Es wird kein offizielles Porsche-Logo verwendet.
Das PHW-Zeichen ist ein eigenes Projektlogo für "Porsche Heritage Wiki".
